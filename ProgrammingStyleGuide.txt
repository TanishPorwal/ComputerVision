The name of the functions inside namespace or global functions begin with capital letter and the subsequent names also start with capital letter. Ex - GaussianBlur
The name of the functions inside class or struct begins with small letter and the subsequent names start with capital letter. Ex - gaussianBlur
The name of variable in class or struct start with m_X, where X is a capital letter and the subsequent names also start with capital letter. Ex - m_Data
Avoid global variables
Use constexpr for constants constants
All the functions have to be in the cv namespace except those in 'SIMD' folder
